# Portfolio — Юлія Мостова

Моє особисте портфоліо: список pet-проєктів, технології, контакти та форма зворотного зв’язку.

### Технології
- React (Hooks, Context)  
- Vite  
- Firebase (для форми зворотного зв’язку)  
- Responsive дизайн (flex, grid)  
- CSS Modules  
- Git / GitHub

### Функціонал
- Перегляд моїх pet-проєктів з демо та посиланнями на GitHub  
- Форма «Написати мені» (відправка листа через Firebase)  
- Адаптивна верстка (мобілка/десктоп)  
- Темна/світла тема (Context)

  ### Демо
https://julimostova.github.io/myPortfolio/

## Запуск локально
```bash

git clone https://github.com/juliMostova/myPortfolio.git
cd myPortfolio)
